Wariant A — Natywne konektory (per platforma)

GA4 przez natywny konektor google-analytics-mcp — jeden OAuth/Service Account widzi wszystkie property gdzie konto jest Viewerem. Meta Ads, Clarity i Shopify dochodzą osobnymi natywnymi konektorami.

GA4 Meta Ads (osobno) Clarity (osobno) Shopify (osobno)

Co dostajesz

✓GA4 — raporty, zdarzenia, lejki, segmenty (wszystkie property po jednym koncie)
✓Meta Ads przez natywny konektor meta-ads-glov
✓Microsoft Clarity przez dedykowany MCP (token)
✓Shopify przez oficjalny Shopify MCP
○Search Console / Google Ads — osobne konektory community (gdy potrzebne)
Dobry start

Wariant B — Oficjalne MCP od Google i Meta

Każde narzędzie loguje się niezależnie przez OAuth. Działa tylko w Claude Desktop (Cowork) — przez Settings → MCP Servers → Add. Bez instalacji NPM.

GA4 Meta Ads

Co dostajesz

✓Meta Ads — 29 narzędzi (kampanie, insighty, katalogi)
✓Prosta instalacja — tylko Settings → Add URL
○Search Console — brak oficjalnego MCP od Google
○Google Ads — brak oficjalnego MCP od Google
○Tylko Cowork (Claude Desktop), nie Terminal
Zaawansowany

Wariant C — Pełna kontrola (własne tokeny)

Każde narzędzie podłączone przez własny token API lub service account. Dla klientów z IT, którzy chcą pełnej kontroli nad dostępami.

GA4 Search Console Google Ads Meta Ads Clarity Shopify

Co dostajesz

✓Pełna kontrola nad tokenami i uprawnieniami
✓Działa w Terminal (Claude Code) i Desktop
✓Można ograniczyć dostęp (read-only)
○Wymaga Google Cloud project + service account
○Skomplikowana konfiguracja — wymaga dewelopera
Dla większości klientów agencji → Wariant A (natywne konektory). GA4 jednym kontem Google (Viewer) widzi wszystkie property — zero warstwy pośredniej. Meta Ads wymaga osobnej autoryzacji przez Facebook, ale to też jeden klik.
Krok po kroku dla każdego źródła

🔵 GA4 — Google Analytics 4

Najlepsza metoda: natywny konektor google-analytics-mcp (aplikacja Google Cloud "eel-analytics") — działa natychmiast, testowane na GLOV.co (Property ID: 342844459)
1

Konektor google-analytics-mcp jest już podłączony

Napisz do Claude'a: "Uruchom raport GA4 dla [nazwa projektu]" — Claude użyje mcp__google-analytics-mcp__run_report

2

Dodaj konto klienta jako Viewer (jeśli nowy klient)

Klient dodaje mail agencji jako Viewer w GA4 → Admin → Property Access Management. Property pojawi się w get_account_summaries od razu — bez nowej konfiguracji konektora.

3

Znajdź Property ID

GA4 Admin → Property Settings → Property ID. Podaj je Claude'owi: "Property ID: 342844459"

Co możesz robić: raporty sesji, transakcji, konwersji, lejki per kanał, segmenty, trendy, porównania okresów, anomalie ruchu

🟢 Google Search Console

Osobny konektor community — gdy potrzebne dane SC, podłącz dedykowany MCP (np. binary ncosentino lub suganthan-gsc-mcp) przez OAuth domeny
1

Sprawdź czy konto Google ma dostęp do Search Console

search.google.com/search-console → zaloguj się kontem klienta → sprawdź czy widzi domeny

2

Podaj domenę Claude'owi

Napisz: "Pobierz dane Search Console dla glov.co z ostatnich 28 dni" — Claude wywoła narzędzie konektora SC

Co możesz robić: top queries, pozycje, kliknięcia, CTR, Core Web Vitals, indeksowanie, błędy crawlowania, mobile usability

🟣 Microsoft Clarity

Dedykowany MCP — wymaga Data Export Token, 10 zapytań/dzień limit
1

Pobierz Data Export Token

clarity.microsoft.com → Settings projektu → Data export → Generate token → skopiuj token

2

Dodaj do claude_desktop_config.json

// ~/Library/Application Support/Claude/claude_desktop_config.json { "mcpServers": { "clarity": { "command": "npx", "args": ["-y", "@microsoft/clarity-mcp-server"], "env": { "CLARITY_API_TOKEN": "TWOJ_TOKEN_TUTAJ" } } } }
3

Restart Claude Desktop

Zamknij i otwórz Claude Desktop — Clarity pojawi się jako dostępne narzędzie

Co możesz robić: heatmapy, nagrania sesji, ścieżki kliknięć, rage clicks, scroll depth, dead clicks, funnel analysis

🔴 Google Ads

Osobny konektor — oficjalny google-ads-mcp (developer token) lub community cohnen/mcp-google-ads. GLOV: czeka na dostęp API do konta
1

Sprawdź Customer ID

ads.google.com → lewy górny róg → Customer ID (format: XXX-XXX-XXXX). GLOV: 169-845-2663

2

Podaj Customer ID Claude'owi

Napisz: "Pobierz wyniki kampanii Google Ads, Customer ID: 169-845-2663"

Co możesz robić: wyniki kampanii, grupy reklam, słowa kluczowe, ROAS, CPA, porównania, optymalizacje stawek

Rekomendacje dla typowych klientów
Startup / SMB

google-analytics-mcp + Clarity MCP

GA4 przez natywny konektor (konto Viewer). Clarity przez NPM token. Opcjonalnie Meta Ads przez konektor meta-ads-glov. Gotowe w <30 min.

E-commerce

google-analytics-mcp + Shopify MCP + Meta

Pełne połączenie: analytics (GA4 przez natywny konektor), sklep (Shopify MCP), reklamy (Meta Ads przez meta-ads-glov lub oficjalny MCP). Pełen obraz, każda platforma natywnie.

Enterprise / Agencja

Własne service accounts (per klient)

Każdy klient ma własny service account Google dodany jako Viewer do swojej property GA4. Izolacja przez property_id w zapytaniu. Pełna kontrola, audytowalne, headless dla automatyzacji.
