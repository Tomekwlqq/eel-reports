0. Opis kategorii

Ekspres do kawy — urządzenie AGD do parzenia kawy (espresso, filtrowanej, kapsułkowej) w domu lub biurze. Klient wchodzi w kategorię rzadko (zakup trwa 3–10 lat) — przy pierwszym zakupie, wymianie zepsutego sprzętu lub chęci przejścia na wyższy poziom kawy.

0b. Problemy, które kategoria załatwia

🎯 Primary (rdzeń — po co kategoria powstała):

Parzy kawę. Dostarcza gotowy napój kawowy bez wychodzenia do kawiarni. Tyle.

Secondary (rynek pociął rdzeń „parzenie kawy" na węższe pod-potrzeby):

  • rano w biegu / bez wysiłku — kawa ma być gotowa szybko, bez znajomości baristyki → ekspres kapsułkowy / w pełni automatyczny
  • espresso jak z kawiarni — chcę jakościowe espresso z prawdziwej kawy mielonej → ekspres kolbowy (semi-auto) z portafiltrem
  • napoje mleczne (cappuccino, latte, flat white) — nie wystarczy samo espresso, potrzebuję pianki → spieniacze + fully-auto z mlecznikiem
  • dla biura / większej grupy — kilkanaście kaw dziennie, niski koszt per kawa, łatwa obsługa → ekspres filtrowy large-batch lub kapsułkowy office
  • mała kuchnia / brak miejsca — kompaktowe urządzenie, nie za duże → rozmiar / footprint jako constraint
  • łatwość i zero obsługi — nie chcę uczyć się spieniania, mielenia, ustawiania ciśnienia → fully automatic, one-touch
  • pierwszy ekspres / wejście w kawę — odchodzę od Nescafé, nie wiem od czego zacząć → prosty onboarding, kapsułki jako most
  • wymiana zepsutego ekspresu — stary padł, szukam zamiennika (ryzyko: lock-in kapsułkowy) → porównanie systemów
  • prezent dla kawoszy — kupuję coś wartościowego dla kogoś, muszę nie wyjść na laika → WOW factor, opis zestawu

⚠️ „Ciśnienie 15 bar", „pojemność 1.8L", „czas nagrzewania 25s" — NIE problemy. To atrybuty. Idą do filtrów/KCF.

Ścieżka zakupowa ma dwa wyraźne garby: stan 3 (CEP) i stan 5 (Preselekcja) — oba ● H — co oznacza, że klient spędza najwięcej energii na dwóch fundamentalnych pytaniach: „który TYP ekspresu pasuje do mojego życia" (routing kapsułkowy/automat/kolbowy) oraz „który konkretny model w tym typie wybrać". Stąd 50–60% budżetu treści powinno obsługiwać routing CEP i preselekcję modelu, a nie generyczne opisy marki. Asymetria wysoki ryzyko/aspiracja (0.45/0.55) przy niskiej pilności (0.25) sygnalizuje, że klient nie kupi pod presją czasu — kupi gdy poczuje że „rozumie co robi", więc treść musi budować pewność decyzji, nie CTA-pressure.

8. Consumer Decision Tree

[L1 — Routing: jaki typ ekspresu?]

Q1. Co ważniejsze — wygoda czy jakość kawy?
  consumer_language: "Chcę coś prostego" vs "Zależy mi żeby kawa była naprawdę dobra"
  → Wygoda (nie chcę się bawić) → [Ekspres kapsułkowy] → Q1-L2-kapsulkowy
  → Jakość + komfort → [Ekspres automatyczny] → Q1-L2-automat
  → Jakość + chcę się nauczyć → [Ekspres kolbowy] → Q1-L2-kolbowy
  → Dużo osób / biuro → [Ekspres filtrowy large-batch] → Q1-L2-filtrowy

Q2. (jeśli niejednoznaczne) Ile osób pije kawę dziennie?
  → 1-2 osoby → compact, zbiornik <1.5L
  → 3-5 osób → średni, 1.5-2L
  → 5+ osób → office tier, podgrzewacz karafki

[L2 — Ekspres kapsułkowy]
Q1. Masz już kapsułki konkretnego systemu?
  consumer_language: "Mam kapsułki Nespresso od kolegi i chcę kupić ekspres"
  → TAK (Nespresso) → tylko Nespresso-compatible
  → TAK (Dolce Gusto) → tylko DG-compatible
  → NIE → wybór systemu: Nespresso Original (espresso/lungo) vs Vertuo (porcjowane)

Q2. Czy chcesz też mleko / cappuccino?
  → Tylko espresso/lungo → Original Line Nespresso bez spieniacza
  → Chcę mleko → Nespresso Lattissima LUB Vertuo LUB Dolce Gusto Genio S Touch

Q3. Design i rozmiar ważne?
  → Mała kuchnia (<30 cm szerokości) → compact footprint
  → Design ważny → premium finish (Creatista, Artisan)

mandatory_in_PDP: system kapsułek + kompatybilność + pojemność zbiornika [L] + wymiary [cm] + ciśnienie [bar]

[L2 — Ekspres automatyczny (fully auto)]
Q1. Jaki smak kawy? Mocne espresso / długa kawa czarna / mix
  consumer_language: "Piję głównie cafe crema a partner ristretto"
  → Mix profili → regulacja stopnia mielenia + programowalne profile
  → Tylko espresso → prostszy model, stały stopień mielenia

Q2. Chcesz podgrzewane mleko / piankę?
  → TAK → zintegrowany spieniacz (carafe) lub LatteCrema
  → NIE → bez spieniacza, niższy model

Q3. Ziarno czy czasem mielona (np. decaf)?
  → Tylko ziarno → standardowy hopper
  → Czasem mielona → bypass hopper (dual use)

Q4. Programowanie / app?
  → TAK → smart connectivity, profile per użytkownik
  → NIE → prostsze UI, niższa cena

Q5. Budżet?
  → 800-1500 zł → entry (DeLonghi Magnifica, Philips 3200)
  → 1500-3000 zł → mid (Eletta Explore, Melitta Barista)
  → 3000+ zł → premium (Jura, DeLonghi La Specialista)

mandatory_in_PDP: typ młynka + liczba poziomów mielenia + spieniacz TAK/NIE + pojemność zbiornika [L] + liczba profili

[L2 — Ekspres kolbowy (semi-auto espresso)]
Q1. Masz już młynek do kawy?
  consumer_language: "Kupuję tylko ekspres" vs "Szukam zestawu ekspres + młynek"
  → TAK → fokus na ekspres, sprawdź kompatybilność portafiltra (58mm standard)
  → NIE → potrzebujesz zestawu lub ekspresu z wbudowanym młynkiem

Q2. Chcesz robić espresso i parować mleko równocześnie?
  consumer_language: "Żona pije cappuccino a ja espresso, chcę obie naraz"
  → TAK → dual boiler (>3000 zł) lub heat-exchange
  → NIE (kolejno) → single boiler lub thermoblock

Q3. Poziom doświadczenia?
  → Początkujący → automatyczna tamperka + gotowe profile ciśnienia
  → Zaawansowany → regulacja ciśnienia / temperatury / pre-infusion

Q4. Ciśnienie pompy?
  → Minimum 15 bar (standard), optimum 18-20 bar dla espresso

mandatory_in_PDP: ciśnienie pompy [bar] + typ grzałki (termoblock/single/dual boiler) + rozmiar portafiltra [mm] + wymiary [cm]
