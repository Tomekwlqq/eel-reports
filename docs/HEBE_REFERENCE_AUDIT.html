Mocne strony (TOP 5)

  1. Tech stack klasy enterprise: SFCC + UseInsider + SF Einstein + Luigi's Box + Criteo + DV360 — dojrzały stack personalizacyjny i reklamowy
  2. Program lojalnościowy My Hebe: ~4M members, diamond tiers — first-party data o wartości Retail Media Network. Analogia Ulta Beauty (USA)
  3. Content commerce blog: 18 kategorii, commerce-linked artykuły → produkty. "Składniki kosmetyków" = SEO powerplay
  4. 316 content slotów HP: Rozbudowana SFCC CMS architektura z wieloma wym. miejscami reklamowymi — gotowość do RMN
  5. Flowbox UGC + VIRAL SHOP: Shoppable social content i influencer-driven sekcja — nowoczesne podejście do discovery

Krytyczne luki (TOP 5)

  1. 2x GA4 ID: G-LRXBRD7NT1 + G-480R3PZWEM — potencjalny split danych analytics lub duplikat. Priorytet 1
  2. Analytics gap — 3 narzędzia własne: UseInsider / Luigi's Box / Flowbox mają własne eventy — nie trafiają do GA4. Blind spots w funnel analysis
  3. Brak filter_select + nav_click events: Nie wiemy jak shopper nawiguje — krytyczna luka dla CRO
  4. Blog → PDP tracking gap: Artykuły linkują do produktów ale klik nie jest trackowany — nie wiemy ile konwersji generuje blog
  5. Brak oznaczeń "Sponsored" na listingu (Criteo + UseInsider) — potencjalne ryzyko compliance UOKiK

Quick Wins (TOP 5)

  1. GTM trigger na klik linków nawigacyjnych → nav_click event w GA4 — 1 godzina pracy
  2. GTM trigger na Luigi's Box select_item → search_click event — integracja LBX → GA4 dataLayer
  3. Weryfikacja 2x GA4 ID — zdeduplikować lub udokumentować multi-property setup
  4. Omnibus price visibility audit — ustawa o cenach wymaga widoczności najniższej ceny 30 dni na kafelku
  5. Blog → PDP: dodać UTM lub custom event na kliknięcia produktów w artykułach
📊 Top 3 obserwacje o hebe.pl (pierwsze wrażenia)
1. Retail Media Network w stadium embrionalnym. Hebe ma wszystkie składniki RMN-tier-1: 4M loyality members, 780 sklepów, pełny stack ad-tech (Criteo, DV360, Meta, Bing, RTB House, pixel.wp.pl), UseInsider do personalizacji, 316 content slotów na HP. Brak: transparentnych oznaczeń "Sponsored" na listingu, publicznego Self-Serve ad portalu dla marek. Porównanie: Sephora / Ulta Beauty w USA mają dojrzałe RMN. Hebe = 2–3 lata za Europą Zachodnią.
2. Personalizacja UseInsider vs. A/B testing — konflikt. 30–35 elementów UseInsider per strona (lista, PDP) = każda sesja jest inaczej wyglądającą stroną. To utrudnia: standardowe screenshoty audytu, CRO testy A/B (interferuje z GTM Experiments), quality assurance UI. Trade-off: personalizacja kosztem powtarzalności UX.
3. Content commerce = wyróżnik konkurencyjny. Blog z 18 kategoriami + "Składniki kosmetyków" + linkowanie artykuł → produkt to rzadka strateg na polskim rynku beauty (Rossmann, Notino nie mają tak development content funnela). SEO value + lower-funnel intent content = długoterminowy asset.
